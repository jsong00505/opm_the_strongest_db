# opm_the_strongest_db

This application is for DB of OPM: The Strongest. It will be displayed in the language of your device(Currently, Korean and English are supported).

## Getting Started

* Clone the repo
* Open the project in Android Studio
* Open pubspec.yaml
* Click `Pub get`
* Run `main.dart`
